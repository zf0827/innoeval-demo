# Showcase

This directory contains supplemental code released together with the open demo.

- `innoeval-search/` keeps a standalone version of the search stack, including the KG, S2, merge, and reranking paths.
- It is separate from the runnable `run_demo.py` flow in the demo root.
- Running the search showcase requires its own environment setup, including Neo4j and the expected graph schema and indexes.
